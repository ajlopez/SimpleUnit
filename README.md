# SimpleUnit

Simple test framework for JavaScript/Node.js

## Installation

Via npm on Node:

```
npm install simpleunit
```


## Usage

Reference in your program:

```js
var sf = require('simpleunit');
```

## Development

```
git clone git://github.com/ajlopez/SimpleUnit.git
cd SimpleUnit
npm install
npm test
```
